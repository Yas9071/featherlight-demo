Numi

Numi was made to bridge the gap between abstract numbers and physical quantity. It is an interactive toolkit that uses camera-based hand tracking to turn math into a tangible experience.

Why Numi Matters

Many children struggle to understand numbers because they are often taught as abstract symbols rather than real concepts. Numi bridges this gap by turning numbers into physical, interactive experiences. By using hand gestures and visual feedback, learners can connect movement with meaning, making math feel more natural and less intimidating.

Key Features

• Gesture-based number input using hand tracking

• Real-time visual feedback

• Simple and intuitive interface for children

• Interactive learning through play

Design Rationale

I chose the Featherlight JavaScript plugin because it creates a simple lightbox interaction that allows users to click on images and view them in a larger format. This works well for Numi because the project is visual, and the plugin helps present interface screens clearly without making the page feel crowded.

The plugin benefits this proof-of-concept by improving the user experience. Instead of placing large images directly on the page, Featherlight allows the gallery to stay clean while still giving users the option to explore each screen in more detail.

Beyond this proof-of-concept, this plugin could be used in a full Numi website to showcase prototype screens, learning activities, product features, or student progress examples. It could also support storytelling by letting users explore different stages of the learning experience one screen at a time.
